Every response should be a bulleted list or a markdown table. No prose paragraphs.

**Why:** Scannable, decision-ready, no filler.

**How to apply:**
- Default: bullets. Comparisons/decisions: tables. Code/commands: fenced blocks unchanged. Status: bullets. Multi-step plans: numbered bullets.
- Never: walls of paragraph prose.

Combine with [[user-single-paste-prompts]]. (Note: caveman mode, when active, is a stricter superset of this.)
